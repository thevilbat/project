# Проект: Прогнозирование стоимости страховых выплат

## Описание проекта

Цель проекта — разработать модель машинного обучения, которая предсказывает **итоговую стоимость страхового возмещения** (`UltimateIncurredClaimCost`) на основе характеристик работника и начальной оценки страхового случая.

Проект включает полный цикл машинного обучения: загрузку и предобработку данных, обучение нескольких моделей регрессии, оценку их качества, анализ важности признаков и создание удобного интерактивного веб-приложения.

**Результат работы:** Многостраничное Streamlit-приложение с возможностью обучения моделей и получения предсказаний в реальном времени.

---

## Датасет

Используется открытый датасет **Workers Compensation** (OpenML ID: **42876**).  
Содержит **100 000 записей** о страховых случаях работников с 14 признаками и одной целевой переменной.

**Ключевые признаки:**
- `Age`, `Gender`, `MaritalStatus`, `WeeklyPay`
- `InitialCaseEstimate` — начальная оценка стоимости случая
- `ClaimDescription` — описание травмы
- `DateTimeOfAccident`, `DateReported` и др.

**Целевая переменная:** `UltimateIncurredClaimCost` (итоговая сумма выплаты).

## Установка и запуск

### 1. Клонирование репозитория

winget install --id Git.Git -e --source winget|

git clone https://github.com/thevilbat/workers_compensation_project.git

cd workers_compensation_project

### 2. Создание и активация виртуального окружения (рекомендуется)

Bash
python -m venv .venv
# Windows
.venv\Scripts\activate
# Linux / macOS
source .venv/bin/activate

### 3. Установка зависимостей

pip install -r requirements.txt

Примечание: Если возникнут проблемы с установкой старых версий pandas, используйте обновлённый requirements.txt с современными версиями (pandas>=2.0).

#### 4. Запуск приложения

streamlit run app.py

Приложение откроется в браузере автоматически.

Как работает приложение

Приложение состоит из двух страниц (используется st.navigation):
1. Анализ и модель (основная страница)

Загрузка данных — поддерживает автоматическую загрузку через OpenML, CSV и ARFF форматы.
Предобработка данных:
Извлечение признаков из дат (ReportingDelay, месяц, день недели)
Кодирование категориальных переменных (LabelEncoder)
Масштабирование числовых признаков (StandardScaler)
Удаление идентификаторов заявок

Обучение моделей — обучаются 4 модели регрессии:
Linear Regression
Ridge Regression
Random Forest
XGBoost

Сравнение моделей по метрикам MAE, RMSE и R²
Форма предсказания — позволяет ввести параметры нового страхового случая и получить прогноз итоговой стоимости возмещения.

2. Презентация

Интерактивная презентация проекта с использованием streamlit-reveal-slides
Автоматически подтягивает актуальные результаты обучения (таблица метрик, лучшая модель, топ важных признаков)
Настраиваемые тема, переходы и плагины

Требования к данным

Поместите файл датасета в папку data/:

workers_compensation.csv или
workers_compensation.arff
